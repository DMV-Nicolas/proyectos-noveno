¿Como añadir un proyecto?

Para añadir un proyecto podemos dirigirnos a la parte de "AÑADIR PROYECTO", en esta seccion se mostrara un formulario donde ingresaremos el titulo del proyecto, en la parte de integrantes, si no son 4 integrantes, simplemente dejar en blanco los faltantes, despues tendras que llenar lo que es la descripcion de tu proyecto, esta descripcion es breve y concisa, puedes guiarte con la informacion del "arroz" ubicada debajo de la pagina o intentar rondar entre los 500 caracteres, a continuacion añadiras el enlace de la imagen que quieras añadir, ademas de la url que lleva a tu proyecto, siempre revisar las letras pequeñas, estas nos indican los requisitos de cada campo.

Error 1904

Este comun error se ocasiona cuando nosotros intentamos añadir un proyecto, pusimos toda nuestra informacion del proyecto y lamentablemente se ocasiono este error, este error tiene solucion y es que en la parte de abajo de cada campo hay unos requerimientos, fijemonos mas bien el requerimiento que dice "obligatorio", si nosotros no llenamos un campo obligatorio nos saldra este error.

Error 2376

Este es ya mas raro de poder encontrarlo y tambien tiene que ver con los requerimentos debajo de cada campo, y este tiene que ver con la longitud maxima de caracteres seleccionados, y es que si podemos ver hay algunos campos que lo requieren como los nombres que permiten hasta 50 caracteres o la descripcion que permite 800.

Error 5489

Este error corresponde a la longitud minima de caracteres, lo contrario a el anterior, este solo se encuentra presente cuando vamos a escribir nuestra descripcion, la cual necesita minimamente 50 caracteres de informacion, asi que no te preocupes, simplemente alarga tu descripcion.

Error 404

Famoso error 404, este error sucede cuando nosotros intentamos acceder desde esta pagina a una ruta que no esta definida o no existe, si acaso te sale este error y estas seguro de estar en una ruta existente porfavor contactarnos desde "CONTACTO", estaremos encatados de responderte :3.
